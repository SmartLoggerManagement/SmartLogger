# Utilisation du fichier .gantter

* Rendez vous sur le site suivant : [http://gantter.com/](http://gantter.com/)<br>
* Cliquer sur le bouton `Start Now`.<br>
* Vous arriverez sur la page de gannter, il vous suffira ensuite d'ouvrir l'onglet `Projet`, puis `Ouvrir` et de sélectionner le fichier .gantter afin d'ouvrr le Gant du projet.
